DESENVOLVIMENTO WEB

Projeto Proffy - Next Level Week

Este foi o meu primeiro projeto desenvolvido na área de programação web. Foi realizado em Abril de 2020 no bootcamp Next Level Week, oferecido pela Rocketseat. O objetivo desse site é a criação de um ambiente educativo possibilitando o agendamento de aulas, de acordo com a disponibilidade do professor e as matérias a serem lecionadas. As linguagens utilizadas foram Javascript, bem como sua aplicação Node.js, assim HTML5 e CSS3, bem como o SQL para programar o banco de dados SQLite.

Projeto Web Design Master -
Danki Code

Este projeto foi desenvolvido no curso Web Design Express oferecido pela Danki Code. Ele envolve apenas a parte visual do site, com o objetivo de colocar em prática conhecimentos de CSS3 e HTML5 relacionados a posicionamento de imagens, criação de formulários, entre outros.

Projeto Sci2Biz -
Iniciação Científica UFRN

Este é o meu primeiro grande site que está sendo feito de forma autônoma. É o meu projeto de iniciação científica da UFRN que ainda está em desenvolvimento, que tem como objetivo a criação de uma plataforma de divulgação científica, bem como a realização dos trabalhos da disciplina de GECTI. Assim, pretende-se a longo prazo fazer desse site um instrumento a fim de aumentar a visualização e importância das pesquisas científicas no Brasil

ARTIGOS E PROJETOS

Projeto - Automação de uma Estação de Tratamento de Esgoto

Este foi um projeto desenvolvido no curso de Automação Industrial do Instituto Metrópole Digital. Fiquei responsável por fazer o projeto de instrumentação e sensores do tanque de cloração e do valo de oxidação da estação de tratamento de esgoto da UFRN. Pode-se observar no projeto componentes como controladores lógicos programáveis (CLP), válvulas e sensores.
